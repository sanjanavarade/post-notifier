== Description ==
A simple plugin to send an email notification to the admin when a blog post is published.

== Installation ==
1. Upload the `post-notifier` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings â†’ Post Notifier and enable notifications.
